Completely sets up a mesh and its manifolds.

This function reads and constructs a mesh according to the given mesh parameters, sets up manifolds on boundaries, and applies boundary conditions. It is intended for distributed triangulations.

Template Parameters
dimThe topological dimension of the mesh.
spacedimThe embedding space dimension (default = dim).
Parameters
[in,out]triangulationThe triangulation to which the grid will be attached.
[in]mesh_parametersParameters defining the mesh or primitive to use.
[in]manifolds_parametersInformation about the manifolds attached to boundaries.
[in]restartFlag indicating whether this is a restart.
[in]boundary_conditionsBoundary condition information, used for periodicity.

Completely sets up a mesh and its manifolds for rotor-stator domains.

The triangulation initially contains the stator triangulation, which is later replaced by the merged rotor-stator triangulation.

Template Parameters

spacedimThe embedding space dimension (default = dim).
Parameters

[in]manifolds_parametersInformation about the manifolds attached to boundaries.
[in]restartFlag indicating whether this is a restart.
[in]boundary_conditionsBoundary condition information, used for periodicity.
[in]mortar_parametersParameters controlling the mortar method, including rotor mesh info.
