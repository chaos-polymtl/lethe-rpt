Detailed Description

Simulation control classes for managing time-stepping and iteration control in Lethe.

This file defines the simulation control hierarchy used in Lethe to manage steady-state and transient simulations. The SimulationControl base class and its derived classes handle time-stepping methods (including BDF and SDIRK), adaptive time stepping, output control, and simulation progression logic.
